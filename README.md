# Vehicle-Detection-using-CNN-Custom-Object-Detection-Model-
This project focuses on building a custom object detection model using Convolutional Neural Networks (CNN) to accurately detect vehicles in images. The workflow includes dataset preparation, model design, training, evaluation, and inference.


---

## 🎯 Objectives

* Prepare a structured dataset for object detection
* Design and train a custom CNN-based detection model
* Evaluate model performance on test data
* Perform inference on sample images to validate detection accuracy

---
